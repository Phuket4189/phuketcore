# BlockServerLauncher (BSL) 项目介绍

## 项目概述

**项目名称**: BlockServerLauncher (BSL)  
**项目简称**: BSL  
**项目类型**: Minecraft服务器启动器  
**开发语言**: Python  
**GUI框架**: PySide6 (Qt6)  
**目标平台**: Windows / Linux / macOS  
**项目状态**: 规划阶段  

---

## 项目背景

### 行业现状

Minecraft作为全球最受欢迎的沙盒游戏，拥有庞大的服务器托管和管理需求。当前市场上的服务器启动器主要分为两类：

1. **客户端启动器**（如HMCL、PCL2）：专注于游戏客户端启动，服务器管理功能相对简单
2. **服务器管理面板**（如Pterodactyl Panel）：功能强大但部署复杂，需要专业的服务器环境

### 市场痛点

**1. 功能分散**
- 客户端启动器缺乏专业的服务器管理功能
- 服务器面板过于复杂，不适合个人用户
- 缺乏一体化的解决方案

**2. 技术门槛高**
- 配置服务器需要专业知识
- Java版本管理复杂
- Mod安装和管理繁琐
- 内网穿透配置困难

**3. 用户体验差**
- 界面设计陈旧
- 操作流程复杂
- 缺乏实时监控
- 多任务处理能力弱

**4. 部署困难**
- 需要复杂的环境配置
- 依赖管理困难
- 跨平台兼容性差
- 缺乏绿色软件特性

### 项目愿景

BlockServerLauncher旨在打造一款**功能强大、用户友好、部署简单**的Minecraft服务器启动器，填补客户端启动器和服务器面板之间的市场空白，为个人用户和小型服务器提供专业级的服务器管理体验。

## 主要价值与特色

### 1. 一体化解决方案
集成服务器启动、管理、监控等功能，无需多个工具配合，统一操作界面，降低学习成本，提高工作效率。填补市场上缺乏一体化解决方案的空白。

### 2. 绿色软件特性
单文件程序，无需安装，自动生成数据文件夹，删除即卸载。参考HMCL、PCL2的成功经验，符合用户使用习惯，降低使用门槛。

### 3. 内网穿透集成
内置FRP内网穿透，一键配置，无需专业网络知识。解决无公网IP问题，降低网络配置难度，节省服务器成本，适合家庭服务器场景。

### 4. 现代化GUI设计
基于PySide6的现代化无边框界面，支持主题切换和动画效果。提升用户体验和操作效率，符合现代审美，增强品牌形象。

### 5. 多任务并行处理
基于asyncio的异步架构，支持多任务并行，不会阻塞界面。解决传统启动器卡顿问题，参考HMCL和Pterodactyl的成功经验，技术架构先进。

### 6. 完善的Mod管理
集成CurseForge和Modrinth API，支持Mod搜索、下载、安装、卸载。简化Mod管理流程，自动处理依赖关系，一站式Mod解决方案。

### 7. 实时性能监控
实时监控系统性能，提供评分系统和优化建议。及时发现性能问题，提供优化参考，预防资源耗尽，提高服务器稳定性。

### 8. 跨平台支持
支持Windows、Linux、macOS三大平台，统一的使用体验。满足不同用户需求，便于服务器迁移，Python天然跨平台，统一代码库。

### 9. 开源免费
完全开源，免费使用，社区驱动发展。降低使用成本，社区贡献功能，透明可审计，持续改进，开源项目更受信任。

### 10. 易于扩展
模块化设计，插件系统，易于扩展新功能。满足个性化需求，社区贡献功能，保持项目活力，适应市场变化，开放架构更有生命力。

**技术优势**：
- 充分利用系统资源
